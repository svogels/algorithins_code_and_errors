🔍 Section 2: Learning About Code Bugs

🔤 Spelling Errors (Syntax Errors)

What it is: A 'Syntax Error' is like a spelling mistake in your code. The computer does not understand it.

What happens: The code will not run at all.

Examples of Spelling Errors:

  • Forgetting a colon (:)
  • Spelling a command wrong (like 'pritn' instead of 'print')
  • Forgetting to close a bracket ()

Example in our Pong Game:

1  # This code has a spelling mistake
2  def move_ball()
3      ball_x = ball_x + 1
4      
5  # The mistake is a missing colon : on line 2

🎯 Let's fix it!

Type line 2 correctly in the box. Add the colon : at the end.

🧠 Wrong Logic Errors

What it is: A 'Logic Error' is when the code runs, but it does the wrong thing. The computer follows your instructions, but the instructions are wrong.

What happens: The game does something you did not want it to do.

Example of a Logic Error:

  • Using a '+' when you should use a '-'
  • Telling the computer to go left when you want it to go right.

Example in our Pong Game:

1  # This code has a logic mistake
2  # When the ball hits the paddle, it should bounce back.
3  # But here, it gets faster and faster! That's wrong.
4  ball_speed = ball_speed * 2
5      
6  # The mistake is using * 2. It should be * -1 to go the other way.

🎯 Let's fix it!

To make the ball bounce back, we need to reverse its speed. Type the correct code in the box.

⚠️ Crashing Errors (Runtime Errors)

What it is: A 'Runtime Error' is an error that happens when you are playing the game. It makes the game crash.

What happens: The game stops working.

Examples of Crashing Errors:

  • Trying to divide a number by zero.
  • Trying to use something that is not there.

Example in our Pong Game:

1  # This code has a crashing error
2  players = ["Player 1", "Player 2"]
3  
4  # Let's try to get the third player. But there is no third player!
5  print(players[2])
6      
7  # The mistake is asking for item 2. In code, we start counting from 0.
8  # So, item 0 is "Player 1" and item 1 is "Player 2". There is no item 2.

🎯 Let's fix it!

To get "Player 2", what number should be in the brackets []? Type the correct code in the box.

📊 Let's Compare The 3 Bugs

Bug Type Does the code run? What happens?
Spelling (Syntax) ❌ No The computer doesn't understand.
Wrong Logic ✅ Yes The game does the wrong thing.
Crashing (Runtime) ⚠️ It crashes! The game stops working.
